# 쿼리 서비스 가이드

이 폴더는 우리가 만든 **쿼리 서비스**가 무엇이고 어떻게 동작하는지를 천천히 설명합니다.
프로그래밍을 막 시작한 사람도 따라올 수 있게 비유 중심으로 썼습니다.

## 읽는 순서

1. [01. 왜 만들었나 — 문제 상황](01-problem.md)
   - 마트 비유로 우리가 왜 이런 서비스를 만들었는지 설명합니다.
2. [02. 어떻게 동작하나 — 큰 그림](02-architecture.md)
   - 시스템의 부품들과 데이터 흐름을 그림으로 봅니다.
3. [03. 주문 목록 예제 따라가기](03-walkthrough.md)
   - 실제 코드가 어떻게 짜여있는지 한 장면씩 봅니다.
4. [04. 안전장치 — DLQ와 멱등성](04-safety.md)
   - "메시지가 깨지면?", "같은 메시지가 두 번 오면?" 같은 상황을 어떻게 막는지.
5. [05. 새 화면 만들기 — 코드 자동 생성](05-new-screen.md)
   - 새로운 화면이 필요할 때 YAML 한 장과 명령어 한 줄로 끝내는 방법.
6. [용어 사전](glossary.md)
   - 자주 나오는 단어들을 한 줄로 정리.
